## 学んだこと
- Gitを戻す場合
  - git restore → 未コミットの変更だけ戻す
  - git reset --hard HEAD → 最後のコミットまで戻す
  - git clean -fd → 未追跡ファイルを削除

## つまずいたポイント・解決方法
- **問題**:変更を戻したいのに戻らない
- **解決策**:未追跡ファイルは restore で戻らない
- **学び**:git clean -fd で完全クリーンアップして解決
